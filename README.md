# Geometry Calculator
A simple geometry calculator with 2D and 3D visualization of flat and solid shapes such as:
* circle
* square
* rectangle
* triangle
* trapezoid
* rhombus
* sphere
* cube
* cuboid
* square pyramid
* cylinder
* cone

Supports area, volume, perimeter, and triangle median calculation.

To run the app use:
```
$ streamlit run app.py
```
